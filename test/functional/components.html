Component load test

The 2D primitives and Texturing are loaded using the components functionality: 


    <x3d components="Texturing,Geometry2D" loadpath="../../src/nodes/" >
    ...

The statements above dynamically load the files ../../src/nodes/Texturing.js and ../../src/nodes/Geometry2D.js. The following options are added:

components
The component to load. Each component needs to be in a .js file.
loadpath
The path or URI prefix where to load the components from.
