Das Logo der iX

Die iX-Geometrie wurde über ein <IndexedFaceSet> realisiert (aus 3ds Max als X3D exportiert), während für die Farbgebung ein selbstdefinerter GLSL-Shader verwendet wurde (s. Quelltext).

attribute vec3 position; attribute vec3 normal; uniform mat4 modelViewMatrix; uniform mat4 modelViewMatrixInverse; uniform mat4 modelViewProjectionMatrix; varying vec3 fragNormal; varying vec3 fragEyeVector; void main() { vec4 eye = vec4(modelViewMatrixInverse * vec4(0.0,0.0,0.0,1.0)); fragEyeVector = eye.xyz - position; fragNormal = normal; gl_Position = modelViewProjectionMatrix * vec4(position, 1.0); } #ifdef GL_ES precision highp float; #endif uniform vec3 diffuseColor; uniform vec3 specularColor; uniform float shininess; varying vec3 fragNormal; varying vec3 fragEyeVector; void main() { vec3 eye = normalize(fragEyeVector); vec3 normal = normalize(fragNormal); vec4 color = vec4(diffuseColor, 1.0); float p = max(0.1, dot(normal, eye)); color.rgb *= p; color.rgb += max(0.1, pow(p, shininess)) * specularColor; gl_FragColor = color; }

Grundsätzliches zur Verwendung von X3DOM

Unterstützte Browser einschließlich einer Anleitung wie man WebGL aktiviert finden sich hier.

Mit Hilfe der Maus kann man die 3D-Objekte von allen Seiten anschauen. Indem man mit gedrückter linker Maustaste den Mauszeiger innerhalb des durch das <x3d>-Element definierten Rechteckes bewegt, kann man die 3D-Szene rotieren. Über die mittlere Maustaste lassen sich die 3D-Objekte verschieben, und mit der rechten Maustaste kann man in die Szene hinein- und hinauszoomen.

Mit der Taste a (vorher das <x3d>-Element durch Anklicken aktivieren) kann man sich alles anschauen, und mit r kommt man wieder zurück an seine Ausgangsposition. Space und d liefern noch einige Zusatzinformationen.

Hinweis: Je nach Qualität der Internetverbindung kann das Laden der 3D-Modelle und Texturen einige Zeit in Anspruch nehmen.
